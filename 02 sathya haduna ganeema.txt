me dharmathawayen karana de, mamai hoo, pudgala widihata gannawa, ( aathma drushtiyen gannawa).

dharmathawaya --> loku galak , loku lidakata atharinawa. dennek prarthanawath karnaawa , gala paawenna kiyala, anika kiyanawa eka yatata yanna kiyala. namuth gala yata giye ai.  me dennagema balapama nisa nevi eka dharmathawayak nisa.  ( nature.)
ex:
bath kanawa - api action eka gana thamai awadaanaya karanna oni, methana action ekathamai, bath kana eka, not about the bath. bath gana hoyanawa nam, kuburata yanakan yanna wenwa, eka iwarayak na. wadak na. 


api samudaya  ( me hataganna deya / origin) kiyala kiyanne me manisikarayen allala dena gathi sobahawayata. samudaya dannawa nam nirodaya therum ganna puluwan.


api danata hoyana widiha
>>api hoyanne siyawath gananakata eha  wechcha dewal gana.
>> dharmayen katha karanne , me dan, me mohothe wena wadak thamai api atheethaya, anagathaya gana plan kaleth me mohothe mai.
>> budun dharmaya deshana karanna thrushnawa nathi karanna. me mohothe sihiyen yukthawa jeewithaya dakinne kohomada kiyana eka thamai budun sakachcha karanne.

wyn-yaatha ( mana aayathana) washayen apa thula yamak samudaya wanne nam,  e pilibadawa, chanda raagaya ain kara ganna , yoniso (hoda paththa) manisikaaraya ( mana ehi karya menehi kirima) pawichchi karanna oni.
ehema kaloth niwana purna wenawa. (meke anith paththa , ayooniso manisikaraya, waradi widihata mana ehi karya meheni kirima)

yoniso manisikaaraya ( anather definition) - sihi path kala yuthu de sihi karnawa , nokala yuthu de sihi nokireema.
ayoniso manisikaaraya- vice versa.

akusalaya - suba nimiththa (wipallaasayak) + ayoniso manasikaraya.
kusalaya - asuba nimiththa + yooniso manasikaaraya

me mohothe wana de ganamai api katha karanne - pachchuppannaya ( prathyen upan) ( dharmathawayan nisa hataganna wenas dharmathawayan  /sadena de) ( meka adyathmaka paththa wiya yuthui, bath ate kathaawa.)
asa explain karanna ba, roopaye upakarayak nathuwa.  asa ta roopayata prathyai ( supported element)
roopaya explain karanna ba asa nathuwa, so  roopaya ta asa prathyai.

asai , roopayai gana katha karanna ba, wyn-yaanaya nathuwa.
wyn-yaanaya poshana labanne asa haa roopaya athi witai.

wyn-yaanaya example : A yakadaya and B yakadaya happawana, is sound coming from A or B ?
wyn-yaanayata names 9k tiyenwa. 

ethakota apita pena dewal okkoma tiyenna nam roopai,asai, wyn-yaanayai thuna ekata pawathinna oni. - api mekata kiyana sprshaya kiyala.  ( thinnan sungathi passo) ( sprsha ayathana 6 mese saade)
me widihata anith ayathana sade,
ithuruwa mama danna oni.
sithiwili (dhamma/ aramunu) + manasa + wyn-yaanayai -> mono sammpassaya .
