1) maggatha / amaggatha sitha ( dyana sith (roopa/aroopa) / kamawachara kusal akusal  )
methan mathaka tiya ganna oni, lokoththara sith ekathu kale na 
ekata hethuwa lokoththara sith widarshanawata ganne nathi nisa 
lokoththara sith hada ganeemata yana margaya thamai meka. 
maggatha/amaggatha kiyala katha karanne yali upadina sith (prathi sandi sahithai)
lokoththara sith me gode di katha karanna ba. 
laukika paththe tiyena sith widarshanaya aran 
lokoththara sitha hata ganeemata hethu sakas kara ganeemak thamai kara ganne. 

2) sauththara / anuththara sitha (
    sauththara - kaama lokaye sith
    anuththara - roopa aroopa sith (widarshanawata ganna puluwan usas dyana sith mewwa)

   widarshanawedi api katha karannema  thrilakshana sobawayan. 
   widarshanawa kiyala kiyanne , saama sankathayakama pawathina 
   podu lakshanayak wana , anithya, dukka , anathma yana 
   thrilakshanayata awaboda kara ganeemai. 
   e kiyanne 
   anithya, dukka ,anathma kiyana dharmatha katha karanna / manasikara karanna puluwan 
   wenne laukika dharama wala pamanai.
   lokoththara dharama wala ebadu manasikarayak na 
   ex: niwana kiwwa dukai, anithyai  kiyala kiyanna ba namuth anathmai. 

   e nisa widarshanawedi lokoththara sitha gana katha karanne na. 
   widarshanawa wadaweema nisa lokoththara sith pahala weemen nirwanaya sakshath wenawa.
   lokoththara sith yanu niwana nevi.
   e nisa arya ashtangika margayama laukikai.  

   lokoththara sith walin thamai duka nathi wenne , ( enam mewa marga sith)
   ehema nowana sith sauththara saha anuthtahra kiyala kotas dekakata beduwa 
   anuththara kiyala kiyanne ewagen usas sith
   roopa / aroopa thala wala athi dyaana sith. 
) 

3) samaahitha sitha / asamaahitha sitha (
    samaahitha - arapana (witharka,wichara,prithi suka, 
    kiyana dyana sith anga walin, witharaka pihitapu sith ) 
    upachara (arpanawata assanna, onna menna) samadi sith
    asamaahita - samadi nathi sitha

    api laga tiyenna kalpanakirim / witharka (neewarana sahitha manasakin karanne)
    dyaana wala di witharka karanne (neewarna yata path karalai)
    e nisa ekata dyana anyak kiyala kiyanawa. 
)
    ehenam api me hithana mattama , ekthara niweemak lesa apita hagewenawa.
    methuwak kal tibunu roopayen alunu daahayan ge adu weemak pamanai sidu wela tiyenne
    e sobawaya thula sanaseemak tiyenawa. eka api dan widinawa
    api eyata alum karanawa, e nisa mattam athara wenasak dakinawa 
    eka kiriyam akaraya awaboda mattmak ehema naththam roopayata eha giya 
    dyaana mattamaka sanaseemak novei.  
    eya keles walin mideemak nevei. 
    ekthara mattamata sanwara bawayak, 
    hikmeema perata wada sakasila, 
    piliwethakata awilla. margaye dorakada asalata 
    emin thamai sitinne. 
    dyaana mattamakata yaama sanada wirya karanna
    sudukam , maanasika watipitawa sakas wemin thamai pawathinne. 
    meya mama wisin karannak nowa 
    meyada hethu prathya samawayen hata gaththa ekak washayen dakiya yuthui. 

4)  wimukththa / awimukththa sitha (
    wimukththa - keles walin miduna , habai thawama lokoththara dharama aramunu karanne na

    thadanga prahanaya - e mohothatai, mohothe keles walin midimak

    thun lokayema tiyena kusal sith mekata aithi wenawa. 

    e nia kusal sith tika thamai wimukththa kiwwe,
    awimukththa kiyanne akusal sith kiyala.
